# WASPS-Scheduling-System

[![Build Status](https://travis-ci.org/mxs4321/WASPS-Scheduling-System.svg?branch=master)](https://travis-ci.org/mxs4321/WASPS-Scheduling-System)


## Setup Instructions

### Server

```bash
cd server
composer install
php -S 0.0.0.0:8000
```

#### Test

```bash
cd server
composer install
composer test
```

### Client

```bash
cd client
npm install
npm start
```

#### Storybooks

```bash
cd client
npm install
npm run storybook
```
