# Setup Instructions

[![Build Status](https://travis-ci.org/BLamy/WASPS-Scheduling-System.svg?branch=master)](https://travis-ci.org/BLamy/WASPS-Scheduling-System)

## Server

```bash
cd server
composer install
php -S 0.0.0.0:8000
```

### Test

```bash
cd server
composer install
composer test
```

## Client

```bash
cd client
npm install
npm start
```

### Storybooks

```bash
cd client
npm install
npm run storybook
```
